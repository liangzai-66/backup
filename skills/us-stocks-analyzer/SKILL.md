# US Stocks Analyzer - 美股分析技能

## 功能

为指定的美股股票提供深度分析报告，包括：
- 最新消息/催化剂
- 基本面分析（财报、营收、预期）
- 技术面分析（支撑/阻力、趋势）
- 期权数据分析（IV、Put/Call 比率、最大痛点）
- 操作建议（股票评级 + 期权策略）

## 使用方法

### 通过 Cron 任务调用

Cron 任务中指定 task 内容，包含：
1. 股票代码列表
2. 分析维度要求
3. 输出格式要求
4. 目标推送渠道

### 分析流程

1. **搜索最新消息** - 使用 `web_search` 或 `searxng` 技能搜索每只股票的最新新闻
2. **获取实时数据** - 访问财经网站获取股价、涨跌幅等实时数据
3. **基本面分析** - 搜索财报、营收、分析师预期等
4. **技术面分析** - 搜索技术分析、支撑阻力位
5. **期权数据** - 搜索期权链、IV、Put/Call 比率、最大痛点
6. **综合分析** - 整合所有信息，给出操作建议

## 数据源推荐

- Yahoo Finance: https://finance.yahoo.com/quote/{SYMBOL}
- MarketWatch: https://www.marketwatch.com/investing/stock/{SYMBOL}
- Options Flow: https://www.barchart.com/stocks/quotes/{SYMBOL}/options
- 新闻搜索：使用 searxng 技能搜索 "{SYMBOL} stock news today"

## 注意事项

1. 美股交易时间：北京时间 21:30-04:00（冬令时 22:30-05:00）
2. 盘前推送（16:00）：侧重隔夜消息和盘前期货
3. 盘中推送（22:00）：侧重实时走势和当日新闻
4. 期权数据需要访问专业网站，可能需要 browser 工具
5. 确保使用 searxng 技能进行联网搜索（根据 MEMORY.md 偏好）
