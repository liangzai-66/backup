# 个股分析 - 美股深度分析技能

## 技能概述

**触发关键词**: 「个股分析」、「股票分析」、「分析 XX 股票」、「XX 股票怎么样」

这是一个专业的美股个股深度分析技能，整合基本面、技术面、期权数据和 VIX 市场分析，为每只股票提供明确的操作建议和最优期权组合策略。

## 适用场景

当用户请求分析美股个股时使用此技能，例如：
- "分析 TSLA"
- "HOOD 股票怎么样"
- "给我分析一下 GOOG"
- "个股分析：AAPL"
- "这只股票值得买入吗？"（附带股票代码）

## 分析框架

### 1. 🔔 最新消息/催化剂

**搜索内容**:
- 公司最新公告、财报发布、产品发布
- 行业新闻、政策变化
- 分析师评级调整
- 盘中突发消息（如果是盘中分析）

**数据源**:
- 使用 `searxng` 技能搜索最新信息
- 搜索关键词：`{ticker} stock news today`, `{ticker} earnings`, `{ticker} analyst rating`

### 2. 📊 基本面分析

**分析维度**:
- 最新财报数据（营收、EPS、利润率）
- 同比增长/环比增长
- 分析师预期与实际情况对比
- 下季度/全年指引
- 估值指标（P/E、PEG、P/B、EV/EBITDA）

**数据源**:
- Yahoo Finance、Google Finance、MarketWatch
- 公司投资者关系页面
- SEC  filings（10-K、10-Q）

### 3. 📉 技术面分析

**分析维度**:
- 当前价格与 52 周范围
- 关键支撑位/阻力位
- 移动平均线（20/50/200 日）
- 技术指标（RSI、MACD、Bollinger Bands）
- 趋势判断（上涨/下跌/盘整）
- 成交量分析

**数据源**:
- TradingView、StockCharts
- 搜索：`{ticker} technical analysis`, `{ticker} support resistance`

### 4. 🎯 期权数据分析

**核心指标**:
- Put/Call 比率（整体及按到期日）
- 未平仓合约 (Open Interest) 分布
- 期权成交量 vs 未平仓合约
- 隐含波动率 (IV) 及 IV Rank
- 异常期权活动（大额交易、Sweeps）
- 最大痛点 (Max Pain) 价位
- Gamma 暴露水平 (GEX)
- Call Wall / Put Floor 价位

**数据源**:
- CBOE、Barchart、Options Flow
- Cheddar Flow、Unusual Whales
- 搜索：`{ticker} options flow`, `{ticker} put call ratio`, `{ticker} open interest`, `{ticker} unusual options activity`

### 5. 📈 VIX 市场分析

**分析维度**:
- 当前 VIX 指数水平及趋势
- VIX 期货期限结构（contango/backwardation）
- VIX 与个股隐含波动率对比
- 市场恐慌/贪婪情绪判断
- VIX 对个股期权策略的影响

**数据源**:
- 搜索：`VIX index current`, `VIX futures term structure`, `market sentiment`

## 输出格式

每只股票的分析报告必须包含以下部分：

### 【股票评级】
```
【股票评级】买入/增持/持有/减持/卖出
理由：基于基本面 + 技术面 + 期权数据的综合判断（2-3 句话）
目标价：$XX（时间框架：1-3 个月）
止损位：$XX
```

### 【期权组合策略】⭐ 核心输出

**只给出一个最优策略**，格式如下：

```
【期权组合策略】策略名称（如 Covered Call / Bull Call Spread / Iron Condor / Cash-Secured Put / Collar 等）
操作：Buy/Sell Call/Put @ 行权价$XX，到期日 YYYY-MM-DD，权利金$XX
      （如果是多腿策略，列出所有腿）
最大盈利：$XX（收益率 X%）
最大亏损：$XX（风险 X%）
盈亏平衡点：$XX
策略理由：为什么选择这个策略（结合 IV、趋势、催化剂等，2-3 句话）
退出条件：何时平仓止盈/止损
```

**常用策略选择指南**:
- **震荡/小幅看涨**: Covered Call（持仓者）/ Cash-Secured Put（想建仓者）
- **明确看涨**: Bull Call Spread
- **明确看跌**: Bear Put Spread
- **区间震荡**: Iron Condor
- **财报前大波动预期**: Straddle/Strangle
- **保护持仓**: Collar

### 【持仓建议】

**针对持仓股票**:
```
【持仓建议】
当前成本：$XX
建议：继续持有/增持/减持/清仓
理由：结合成本价给出解套/止盈建议
增收策略：是否适合做 Covered Call（如适合，给出建议行权价）
```

**针对关注股票**（无持仓）:
```
【建仓建议】
建议入场价位：$XX-$XX
建议仓位：总资金的 X%
建仓计划：分批建仓（如：现价建仓 X%，跌至$XX 加仓 X%）
```

### 【风险提示】

```
【风险提示】
- VIX 水平及市场风险评估
- 个股特定风险（财报日、锁定期解禁、诉讼等）
- 期权策略特有风险
```

## 技能使用流程

1. **识别股票代码**: 从用户输入中提取股票 ticker（如 TSLA、AAPL）

2. **确认用户持仓状态**: 
   - 如果用户提供了持仓信息（股数、成本），在建议中结合成本价
   - 如果未提供，按一般分析处理

3. **数据搜集**（使用 searxng 技能）:
   - 搜索最新消息
   - 搜索基本面数据
   - 搜索技术面数据
   - 搜索期权数据
   - 搜索 VIX 数据

4. **综合分析**:
   - 整合所有维度的信息
   - 形成明确的股票评级
   - 选择最优期权策略
   - 给出具体操作建议

5. **输出报告**: 按上述格式输出完整分析报告

## 示例

### 示例 1: 持仓股票分析

```
用户：个股分析 TSLA，我持有 50 股成本$446.91

助手：📈 TSLA (Tesla) - 深度分析

【股票评级】持有
理由：基本面稳健但估值偏高，技术面在$400-500 区间震荡，期权数据显示中性偏多
目标价：$480（1-3 个月）| 止损位：$420

【期权组合策略】Covered Call（备兑看涨）
操作：Sell Call @ $500，到期日 2026-04-18，权利金$15.50
最大盈利：$68.50/股（收益率 15.3%）
最大亏损：股价跌至$0（股票持仓风险）
盈亏平衡点：$431.50
策略理由：IV 较高 (55%)，权利金丰厚；股价在阻力位附近；可降低持仓成本
退出条件：股价突破$510 或 IV 降至 40% 以下平仓

【持仓建议】
当前成本：$446.91
建议：继续持有
理由：当前价位接近成本，建议持有等待反弹
增收策略：适合做 Covered Call，每月可收取$10-20 权利金降低成本

【风险提示】
- VIX 处于 18 中等水平，市场情绪稳定
- 注意财报前 IV 上升风险
- 电动车行业竞争加剧风险
```

### 示例 2: 关注股票分析

```
用户：分析 COIN 这只股票

助手：📈 COIN (Coinbase Global) - 深度分析

【股票评级】增持
理由：加密货币市场复苏，交易量回升，基本面改善，技术面突破关键阻力
目标价：$280（1-3 个月）| 止损位：$220

【期权组合策略】Bull Call Spread（牛市看涨价差）
操作：Buy Call @ $240 + Sell Call @ $260，到期日 2026-04-18
净支出：$8.50
最大盈利：$11.50（收益率 135%）
最大亏损：$8.50（权利金）
盈亏平衡点：$248.50
策略理由：明确看涨趋势，价差策略降低成本和风险；IV 适中
退出条件：股价突破$260 或亏损达 50% 时平仓

【建仓建议】
建议入场价位：$235-$245
建议仓位：总资金的 5-8%
建仓计划：现价建仓 50%，跌至$220 加仓 50%

【风险提示】
- 加密货币价格波动风险
- 监管政策不确定性
- VIX 若大幅上升可能影响科技股估值
```

## 注意事项

1. **数据时效性**: 所有数据应尽可能使用最新数据（当天或最近交易日）

2. **客观平衡**: 呈现多空双方的观点，避免过度乐观或悲观

3. **风险披露**: 必须包含风险提示，特别是期权策略的风险

4. **明确建议**: 评级和建议必须明确，避免模棱两可

5. **结合用户持仓**: 如果用户提供了持仓信息，建议必须结合成本价

6. **期权策略适当性**: 
   - 持仓者：优先推荐 Covered Call、Collar
   - 看涨但无持仓：推荐 Bull Call Spread、Cash-Secured Put
   - 看跌：推荐 Bear Put Spread
   - 震荡：推荐 Iron Condor
